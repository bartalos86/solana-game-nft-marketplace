# Platform Frontend (Ethereum)

This is the Ethereum version of the platform frontend. It is the user-facing marketplace/game web app that talks to the contracts in `eth-project` and uses shared code from the local SDK packages.

## Prerequisites

- Node.js 20+
- pnpm

## Start

```bash
pnpm install
pnpm dev
```

## Tests

This package currently does not define a dedicated frontend test script in `package.json`.
