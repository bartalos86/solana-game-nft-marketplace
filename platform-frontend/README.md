# Platform Frontend (Solana)

This is the main Solana-facing web app for the marketplace and game. It handles the player-facing UI and connects to the Solana programs and shared SDK logic used across the repo.

## Prerequisites

- Node.js 20+
- pnpm

## Start

```bash
pnpm install
pnpm dev
```

## Tests

This package currently does not define a dedicated frontend test script in `package.json`.
