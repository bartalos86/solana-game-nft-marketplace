# Sample Game (Solana)

This is a playable sample client built on the Solana stack. It is used to validate gameplay and marketplace interactions with wallets and the shared SDK in a real frontend flow.

## Prerequisites

- Node.js 20+
- pnpm

## Start

```bash
pnpm install
pnpm dev
```

## Tests

Run all tests:

```bash
pnpm test
```

Run Irys cost test only:

```bash
pnpm run test:irys-cost
```
