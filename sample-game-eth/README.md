# Sample Game (Ethereum)

This is a playable sample client built on the Ethereum stack. It is mainly used to exercise game flows end to end against the shared SDK and marketplace/game services.

## Prerequisites

- Node.js 20+
- pnpm

## Start

```bash
pnpm install
pnpm dev
```

## Tests

Run all tests:

```bash
pnpm test
```

Run Irys cost test only:

```bash
pnpm run test:irys-cost
```
